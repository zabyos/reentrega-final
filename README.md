# Ecommerce Backend

Proyecto final Coderhouse

## Tecnologías
- Node.js
- Express
- MongoDB
- Mongoose

## Funcionalidades
- CRUD de productos
- Carrito de compras
- Paginación, filtros y ordenamiento
- Vistas con Handlebars

## Endpoints principales

### Productos
GET /api/products
POST /api/products

### Carritos
POST /api/carts
GET /api/carts/:cid
POST /api/carts/:cid/products/:pid
